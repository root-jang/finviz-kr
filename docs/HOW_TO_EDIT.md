# 코드 수정 가이드

## 웹 개발의 핵심: Hot Reload (자동 새로고침)

웹 개발의 가장 큰 장점은 **코드를 수정하면 브라우저가 자동으로 새로고침**된다는 것입니다!

## 수정 방법

### 1. 파일 열기
VS Code 왼쪽 파일 탐색기에서 파일 클릭

### 2. 코드 수정
원하는 내용 수정

### 3. 저장
- `Ctrl + S` 또는 파일 → 저장
- **저장하면 자동으로 브라우저가 새로고침됨!**

### 4. 브라우저 확인
변경사항이 즉시 반영됨

## 주요 파일 위치

### 메인 페이지 수정
**파일**: `app/page.tsx`
- 홈페이지 내용 수정
- "FinViz KR" 제목, 설명 등

### 종목 리스트 페이지
**파일**: `app/stocks/page.tsx`
- 종목 리스트 표시 방식 수정

### 종목 상세 페이지
**파일**: `app/stocks/[code]/page.tsx`
- 종목 상세 정보 표시 수정

### 스타일 수정
**파일**: `app/globals.css`
- 전체 스타일 수정

### 네비게이션 수정
**파일**: `components/Navigation.tsx`
- 상단 메뉴바 수정

## 실전 예제

### 예제 1: 홈페이지 제목 변경

1. `app/page.tsx` 파일 열기
2. 다음 부분 찾기:
   ```tsx
   <h1 className="text-3xl sm:text-4xl lg:text-5xl font-bold mb-4 text-gray-900">
     FinViz KR
   </h1>
   ```
3. "FinViz KR"을 원하는 텍스트로 변경
4. 저장 (`Ctrl + S`)
5. 브라우저 자동 새로고침 확인!

### 예제 2: 색상 변경

1. `tailwind.config.ts` 파일 열기
2. `primary` 색상 변경:
   ```typescript
   primary: {
     DEFAULT: '#2563eb', // 이 색상을 원하는 색으로 변경
   }
   ```
3. 저장 후 브라우저 확인

### 예제 3: 카드 내용 수정

1. `app/page.tsx` 파일 열기
2. `features` 배열 찾기
3. 카드 내용 수정:
   ```tsx
   {
     title: '종목 리스트',
     description: '원하는 설명으로 변경',
     href: '/stocks',
     icon: '📊',
   }
   ```

## 개발 팁

### 1. 실시간 확인
- VS Code와 브라우저를 나란히 배치
- 코드 수정 → 저장 → 즉시 브라우저 확인

### 2. 에러 확인
- 코드에 오류가 있으면:
  - 터미널에 빨간색 에러 메시지 표시
  - 브라우저 콘솔 (`F12`)에서도 확인 가능

### 3. 자동 완성
- VS Code가 자동으로 코드 완성 제안
- `Ctrl + Space`로 강제로 자동 완성 표시

### 4. 파일 찾기
- `Ctrl + P`로 파일 이름으로 빠르게 찾기
