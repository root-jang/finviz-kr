# 문제 해결 가이드

## Internal Server Error 해결 방법

### 1단계: 터미널 에러 메시지 확인

**가장 중요!** VS Code 터미널을 확인하세요:
- 빨간색 에러 메시지가 있는지 확인
- 에러 메시지를 복사해서 확인

### 2단계: 개발 서버 재시작

터미널에서:
1. `Ctrl + C`로 서버 중지
2. 다시 실행: `npm run dev`

### 3단계: 브라우저 캐시 삭제

브라우저에서:
- `Ctrl + Shift + Delete` → 캐시 삭제
- 또는 `Ctrl + F5` (강제 새로고침)

### 4단계: .next 폴더 삭제 (캐시 문제)

터미널에서:
```bash
# 서버 중지 후
rmdir /s /q .next
npm run dev
```

## 자주 발생하는 오류

### 문법 오류
- 따옴표 누락
- 괄호 불일치
- 세미콜론 누락

### 타입 오류
- TypeScript 타입 불일치

### 파일 경로 오류
- import 경로 잘못됨
